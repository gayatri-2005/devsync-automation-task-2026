# DevSync Automation Task

This repository is set up with a GitHub Action that performs a daily automated commit to track activity and maintain a consistent history.

## Features
- **Daily Schedule**: Runs once a day via GitHub Actions.
- **Automated Commit**: Updates `activity.txt` with a timestamp.
- **Manual Trigger**: Can be manually run via `workflow_dispatch`.
